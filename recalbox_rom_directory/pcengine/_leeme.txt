=======================================================================
# RECALBOX: Sistema PC Engine (Consola de Salón)
=======================================================================

Copiad vuestras roms PC Engine aquí.
Las extensiones soportadas son: .7z .pce .m3u .zip

Este sistema es emulado por 3 emuladores diferentes.
Estos emuladores se listan a continuación por order de compatibilidad/facilidad de uso:
libretro-mednafen_supergrafx soporta ficheros de tipo .m3u .pce .zip .7z y es compatible netplay.
libretro-mednafen_pce_fast soporta ficheros de tipo .m3u .pce .zip .7z y es compatible netplay.
libretro-fbneo soporta ficheros de tipo .zip .7z y no es compatible netplay.

Periféricos soportados:
- Los mandos son obligatorios !
- Un teclado es no necesario
- Un ratón es no necesario

Este sistema permite usar roms comprimidas en .zip y/o .7z.
Sin embargo solo se trata de un archivo de almacenamiento. Los archivos dentro de .zip/.7z deben coincidir con las extensiones mencionadas arriba.
Cada archivo .zip/.7z debe contener únicamente una rom comprimida.
