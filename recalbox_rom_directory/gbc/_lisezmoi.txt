=======================================================================
# RECALBOX: Système Game Boy Color (Console portable)
=======================================================================

Copiez vos roms Game Boy Color ici.
Les extensions supportées sont: .gb .7z .gbc .bin .zip

Ce système peut être émulé par 5 émulateurs differents.
Ces emulateurs sont listés ci dessous par ordre de compatibilité/facilité d'utilisation :
libretro-gambatte supporte des fichiers de type .gb .gbc .bin .zip .7z et n'est pas compatible netplay.
libretro-tgbdual supporte des fichiers de type .gb .gbc .bin .zip .7z et n'est pas compatible netplay.
libretro-mgba supporte des fichiers de type .gb .gbc .bin .zip .7z et est compatible netplay.
libretro-sameboy supporte des fichiers de type .gb .gbc .bin .zip .7z et est compatible netplay.
libretro-mesen_s supporte des fichiers de type .gb .gbc .zip .7z et n'est pas compatible netplay.

Périphériques supportés :
- Les manettes sont obligatoires !
- Un clavier est inutile
- Une souris est inutile

Ce système supporte également les roms compressées au format .zip/.7z.
Attention toutefois, il ne s'agit que d'une archive.
Les fichiers contenus dans les .zip/.7z doivent correspondre aux extensions citées précédemment.
De plus, chaque fichier .zip/.7z ne doit contenir qu'une seule rom compressée.

Consultez la documentation à cette adresse :
https://wiki.recalbox.com/fr/emulators/handhelds/game-boy-color
