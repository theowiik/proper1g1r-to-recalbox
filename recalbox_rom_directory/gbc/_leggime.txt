=======================================================================
# RECALBOX: Sistema Game Boy Color (Console Portatile)
=======================================================================

Copia qui le rom per Game Boy Color.
I file supportati sono: .gb .7z .gbc .bin .zip

Questo sistema può essere emulato con 5 emulatori.
Tutti gli emulatori sono in questa lista, in ordine di compatibilità/semplicità di utilizzo:
libretro-gambatte supporta i file di .gb .gbc .bin .zip .7z e NON è compatibile con Netplay.
libretro-tgbdual supporta i file di .gb .gbc .bin .zip .7z e NON è compatibile con Netplay.
libretro-mgba supporta i file di .gb .gbc .bin .zip .7z e  è compatibile con Netplay.
libretro-sameboy supporta i file di .gb .gbc .bin .zip .7z e  è compatibile con Netplay.
libretro-mesen_s supporta i file di .gb .gbc .zip .7z e NON è compatibile con Netplay.

Requisiti di sistema:
- Il Controller è obbligatorio
- La Tastiera è non necessaria
- Il Mouse è non necessario

Questo sistema permette di utilizzare rom compresse in formato .zip o .7z.
Ma si tratta solamente di un file d'archivio. I file all'interno del .zip/.7z devono combaciare con le estensioni menzionate in precedenza.
Ogni .zip/.7z deve contenere solamente una rom compressa.
