=======================================================================
# RECALBOX: Sistema AmstradCPC (Computer)
=======================================================================

Copia qui le rom per AmstradCPC.
I file supportati sono: .7z .sna .cdt .kcr .tap .cpr .voc .bin .dsk .m3u .zip

Questo sistema può essere emulato con 2 emulatori.
Tutti gli emulatori sono in questa lista, in ordine di compatibilità/semplicità di utilizzo:
libretro-cap32 supporta i file di .bin .cdt .cpr .dsk .m3u .sna .tap .voc .zip .7z e  è compatibile con Netplay.
libretro-crocods supporta i file di .dsk .kcr .sna e  è compatibile con Netplay.

Requisiti di sistema:
- Il Controller è raccomandato
- La Tastiera è obbligatoria
- Il Mouse è opzionale

Questo sistema permette di utilizzare rom compresse in formato .zip o .7z.
Ma si tratta solamente di un file d'archivio. I file all'interno del .zip/.7z devono combaciare con le estensioni menzionate in precedenza.
Ogni .zip/.7z deve contenere solamente una rom compressa.
