=======================================================================
# RECALBOX: Système Dos (x86) (Ordinateur)
=======================================================================

Copiez vos roms Dos (x86) ici.
Les extensions supportées sont: .dos .pc .zip

Ce système peut être émulé par 2 émulateurs differents.
Ces emulateurs sont listés ci dessous par ordre de compatibilité/facilité d'utilisation :
libretro-dosbox_pure supporte des fichiers de type .zip et n'est pas compatible netplay.
dosbox-dosbox supporte des fichiers de type .dos .pc et n'est pas compatible netplay.

Périphériques supportés :
- Les manettes sont recommandées
- Un clavier est recommandé
- Une souris est recommandée

Ce système supporte également les roms compressées au format .zip/.7z.
Attention toutefois, il ne s'agit que d'une archive.
Les fichiers contenus dans les .zip/.7z doivent correspondre aux extensions citées précédemment.
De plus, chaque fichier .zip/.7z ne doit contenir qu'une seule rom compressée.

Consultez la documentation à cette adresse :
https://wiki.recalbox.com/fr/emulators/computers/dos
