=======================================================================
# RECALBOX: Sistema Nintendo 64 (Console da Tavolo)
=======================================================================

Copia qui le rom per Nintendo 64.
I file supportati sono: .n64 .7z .v64 .z64 .zip

Questo sistema può essere emulato con 4 emulatori.
Tutti gli emulatori sono in questa lista, in ordine di compatibilità/semplicità di utilizzo:
mupen64plus-glide64mk2 supporta i file di .n64 .v64 .z64 e NON è compatibile con Netplay.
libretro-parallel_n64 supporta i file di .n64 .v64 .z64 .zip .7z e NON è compatibile con Netplay.
libretro-mupen64plus_next supporta i file di .n64 .v64 .z64 .zip .7z e NON è compatibile con Netplay.
mupen64plus-rice supporta i file di .n64 .v64 .z64 e NON è compatibile con Netplay.

Requisiti di sistema:
- Il Controller è obbligatorio
- La Tastiera è non necessaria
- Il Mouse è non necessario

Questo sistema permette di utilizzare rom compresse in formato .zip o .7z.
Ma si tratta solamente di un file d'archivio. I file all'interno del .zip/.7z devono combaciare con le estensioni menzionate in precedenza.
Ogni .zip/.7z deve contenere solamente una rom compressa.
