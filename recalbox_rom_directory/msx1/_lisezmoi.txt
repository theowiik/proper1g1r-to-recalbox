=======================================================================
# RECALBOX: Système MSX1 (Ordinateur)
=======================================================================

Copiez vos roms MSX1 ici.
Les extensions supportées sont: .rom .msx1 .7z .cas .ri .mx1 .dsk .m3u .zip

Ce système peut être émulé par 3 émulateurs differents.
Ces emulateurs sont listés ci dessous par ordre de compatibilité/facilité d'utilisation :
libretro-bluemsx supporte des fichiers de type .cas .dsk .mx1 .m3u .ri .rom .zip .7z et n'est pas compatible netplay.
libretro-fmsx supporte des fichiers de type .rom .msx1 .dsk .cas .m3u .zip .7z et n'est pas compatible netplay.
libretro-fbneo supporte des fichiers de type .zip .7z et n'est pas compatible netplay.

Périphériques supportés :
- Les manettes sont recommandées
- Un clavier est recommandé
- Une souris est inutile

Ce système supporte également les roms compressées au format .zip/.7z.
Attention toutefois, il ne s'agit que d'une archive.
Les fichiers contenus dans les .zip/.7z doivent correspondre aux extensions citées précédemment.
De plus, chaque fichier .zip/.7z ne doit contenir qu'une seule rom compressée.

Consultez la documentation à cette adresse :
https://wiki.recalbox.com/fr/emulators/computers/msx-1
