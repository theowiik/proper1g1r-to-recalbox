=======================================================================
# RECALBOX: Sistema ZXSpectrum (Computador)
=======================================================================

Copie suas roms de ZXSpectrum aqui.
As extensões suportadas são: .tzx .7z .trd .scl .zip .slt .tap .csw .z80 .dsk .rzx .szx

Esse sistema podem ser emulados usando 2 emuladores.
Todos os emuladores estão listados abaixo, em ordem de compatibilidade/facilidade de uso:
libretro-fuse suporta arquivos de .csw .dsk .rzx .scl .slt .szx .tap .trd .tzx .z80 .zip .7z e NÃO é compatível com netplay/jogo em rede.
libretro-fbneo suporta arquivos de .zip .7z e NÃO é compatível com netplay/jogo em rede.

Dispositivo necessário:
- Controle é obrigatório
- Teclado é opcional
- Mouse é desnecessário

Este sistema permite usar roms compactadas em .zip e/ou .7z.
Mas apenas como forma de armazenamento. Os arquivos dentro do .zip/.7z devem corresponder às extensões mencionadas acima.
Cada arquivo .zip/.7z deve conter apenas uma rom compactada.
