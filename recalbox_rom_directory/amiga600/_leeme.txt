=======================================================================
# RECALBOX: Sistema Amiga 600 (Ordenador)
=======================================================================

Copiad vuestras roms Amiga 600 aquí.
Las extensiones soportadas son: .fdi .slave .lzx .7z .adf .ipf .hdz .lha .info .raw .adz .rp9 .hdf .dms .m3u .lhz .zip

Este sistema es emulado por 3 emuladores diferentes.
Estos emuladores se listan a continuación por order de compatibilidad/facilidad de uso:
libretro-uae4arm soporta ficheros de tipo .adf .adz .dms .hdf .hdz .ipf .lha .lhz .lzx .m3u .zip y no es compatible netplay.
libretro-puae soporta ficheros de tipo .adf .adz .dms .fdi .hdf .hdz .info .ipf .lha .lhz .lzx .m3u .raw .slave .zip .7z y no es compatible netplay.
amiberry-amiberry soporta ficheros de tipo .adf .ipf .dms .fdi .hdf .hdz .lha .lhz .lzx .rp9 .zip y no es compatible netplay.

Periféricos soportados:
- Los mandos son obligatorios !
- Un teclado es opcional
- Un ratón es opcional

Este sistema permite usar roms comprimidas en .zip y/o .7z.
Sin embargo solo se trata de un archivo de almacenamiento. Los archivos dentro de .zip/.7z deben coincidir con las extensiones mencionadas arriba.
Cada archivo .zip/.7z debe contener únicamente una rom comprimida.
