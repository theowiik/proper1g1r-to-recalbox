=======================================================================
# RECALBOX: Sistema Family Computer Disk System (Consola de Salón)
=======================================================================

Copiad vuestras roms Family Computer Disk System aquí.
Las extensiones soportadas son: .7z .fds .zip

Este sistema es emulado por 4 emuladores diferentes.
Estos emuladores se listan a continuación por order de compatibilidad/facilidad de uso:
libretro-nestopia soporta ficheros de tipo .fds .zip .7z y es compatible netplay.
libretro-fceumm soporta ficheros de tipo .fds .zip .7z y es compatible netplay.
libretro-mesen soporta ficheros de tipo .fds .zip .7z y es compatible netplay.
libretro-fbneo soporta ficheros de tipo .zip .7z y es compatible netplay.

Periféricos soportados:
- Los mandos son obligatorios !
- Un teclado es no necesario
- Un ratón es no necesario

Este sistema permite usar roms comprimidas en .zip y/o .7z.
Sin embargo solo se trata de un archivo de almacenamiento. Los archivos dentro de .zip/.7z deben coincidir con las extensiones mencionadas arriba.
Cada archivo .zip/.7z debe contener únicamente una rom comprimida.

Más información y documentación en:
