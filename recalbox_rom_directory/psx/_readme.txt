# RECALBOX: System Sony Playstation 1 (Home console)
=======================================================================

Copy your Sony Playstation 1 ROMs here.
Supported extensions are: .psexe .exe .ecm .cue .cbn .chd .img .ccd .mds .iso .bin .toc .m3u .pbp .psf .mdf

This system can be emulated using 4 emulators.
These emulators are listed below, in order of compatibility/ease of use:
libretro-pcsx_rearmed supports files of .bin .cbn .ccd .chd .cue .exe .iso .m3u .mdf .pbp .toc and is NOT Netplay compatible.
libretro-mednafen_psx supports files of .ccd .chd .cue .m3u .pbp .toc and is NOT Netplay compatible.
libretro-swanstation supports files of .chd .cue .ecm .exe .img .iso .m3u .mds .pbp .psf .psexe .toc and is NOT Netplay compatible.
duckstation-duckstation supports files of .bin .chd .cue .ecm .img .iso .m3u .mds .pbp and is NOT Netplay compatible.

Device requirements:
- A controller is required
- A keyboard is unusable
- A mouse is unusable


More information and documentation at:
https://wiki.recalbox.com/en/emulators/consoles/playstation-1
