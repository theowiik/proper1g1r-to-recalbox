=======================================================================
# RECALBOX: Sistema Sega SG1000 (Console da Tavolo)
=======================================================================

Copia qui le rom per Sega SG1000.
I file supportati sono: .7z .sg .sc .bin .zip

Questo sistema può essere emulato con 5 emulatori.
Tutti gli emulatori sono in questa lista, in ordine di compatibilità/semplicità di utilizzo:
libretro-genesisplusgx supporta i file di .sg .zip .7z e  è compatibile con Netplay.
libretro-gearsystem supporta i file di .bin .sg .zip .7z e  è compatibile con Netplay.
libretro-fbneo supporta i file di .zip .7z e  è compatibile con Netplay.
libretro-bluemsx supporta i file di .sc .sg .zip .7z e NON è compatibile con Netplay.
libretro-picodrive supporta i file di .bin .sc .sg .zip .7z e  è compatibile con Netplay.

Requisiti di sistema:
- Il Controller è obbligatorio
- La Tastiera è non necessaria
- Il Mouse è non necessario

Questo sistema permette di utilizzare rom compresse in formato .zip o .7z.
Ma si tratta solamente di un file d'archivio. I file all'interno del .zip/.7z devono combaciare con le estensioni menzionate in precedenza.
Ogni .zip/.7z deve contenere solamente una rom compressa.
